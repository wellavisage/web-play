THML tags (HTML тэги)

Общая информация:

  1. Тэг имеет форму <ИМЯ_ТЭГА>
  2. Большинство тегов имеют открывающий <html> и закрывающий тэг </html>, все что находится между ними - сожержимое тэга
  3. Тэг может иметь аттрибуты, например тэг ссылки иммет аттрибут href, который содержит адрес, на который указывает ссылка. Например: <a href="http://google.com">google</a>
  4. Тэги бывают 2х типов, блочные и строчные.
    • Блочный - занимает всю доступную ширину котейнера в котором находится. Например DIV, P, FORM, UL, OL...
    • Строчный - оборачивает вложенный текст, иногда добавляя ему какой-то эффет. Например STRONG, EM, A, SPAN...
  5. Блочные элементы состоят из 4х частей.
    • margin - невидимая внешная граница. Важно понимать что внешняя граница указывает минимальное свободное место между границей одного блочного элемента и границей другого блочного элемента. В случае когда 2 соседних эелемента имеют внешнюю границу, расстояние между ними будет равно большей внешней границе, а не сумме их внешних границ
    • border - видимая граница элемента. Может быть стилизована
    • padding - Внутренняя граница элемента. Используется зачастую чтобы оставить промежуток между текстом и видимой границей
    • box - непомтердстевенно само содержимое элемента. Box имеет высоту height и ширину width
